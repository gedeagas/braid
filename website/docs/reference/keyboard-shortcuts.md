# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Braid.

## Global

| Shortcut | Action |
|---|---|
| **Cmd+,** | Open Settings |
| **Cmd+Shift+M** | Toggle Mission Control |
| **Cmd+B** | Toggle Sidebar |
| **Cmd+Alt+R** | Toggle Right Panel |
| **Escape** | Close dialogs and dropdowns |

## Chat

| Shortcut | Action |
|---|---|
| **Cmd+L** | Focus chat input |
| **Enter** | Send message |
| **/** | Trigger slash command autocomplete |
| **@** | Trigger file mention autocomplete |
| **Arrow Up / Down** | Navigate autocomplete suggestions |
| **Enter** | Select autocomplete suggestion |
| **Escape** | Dismiss autocomplete |

## Editor

| Shortcut | Action |
|---|---|
| **Cmd+S** | Save file |

## Tabs

| Shortcut | Action |
|---|---|
| **Double-click tab** | Rename session or file tab |

## Trackpad Gestures

| Gesture | Action |
|---|---|
| **Two-finger swipe left** | Cycle to the next tab in the center panel |
| **Two-finger swipe right** | Cycle to the previous tab in the center panel |

Trackpad swipe navigation is an experimental feature. Enable it from **Settings > Experimental > Magic Trackpad**. It cycles through whatever is open in the center panel, such as sessions, files, and changes.
