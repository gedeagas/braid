# Claude Plugins

Plugins extend Claude's capabilities with third-party integrations. Braid shows installed plugins and lets you enable or disable them.

## Open plugins settings

Go to **Settings > Claude Config > Plugins**.

## View installed plugins

Each plugin displays:

- **Name** - The plugin identifier.
- **Version** - The installed version.
- **Scope** - Whether the plugin is global or project-scoped.
- **Status** - Enabled or disabled.

## Enable and disable plugins

Toggle a plugin on or off using the switch next to its name. Changes take effect on the next session start. Running sessions continue with their original plugin configuration.

:::note
Plugin installation is managed outside of Braid through the Claude CLI. This settings page only controls which installed plugins are active.
:::
