# Appearance

Open Settings with **Cmd+,** and select the **Appearance** tab to adjust zoom level, choose a theme, or import custom themes.

## Options

| Option | Description | Details |
|---|---|---|
| **UI Zoom** | Scale the entire interface up or down. Use the **+** and **-** buttons to adjust in increments, or click **Reset** to return to 100%. | Range varies by display |
| **Theme** | Pick a color theme for the app. | Default, Dark, Light, Accessibility, or any imported custom theme |
| **Delete custom themes** | Remove any previously imported custom theme. Built-in themes cannot be deleted. | Available per custom theme |

## Built-in themes

| Theme | Description |
|---|---|
| **Default** | Adapts to your system appearance (dark or light). |
| **Dark** | Fixed dark background with light text. |
| **Light** | Fixed light background with dark text. |
| **Accessibility** | High-contrast colors optimized for readability. |

## Tips

- Zoom adjustments persist across sessions.
- If text appears too small on a high-DPI display, increase the zoom level rather than changing your system display scaling.
