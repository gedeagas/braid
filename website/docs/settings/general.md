# General

Open Settings with **Cmd+,** and select the **General** tab to configure language, activity messages, and confirmation dialogs.

## Options

| Option | Description | Values |
|---|---|---|
| **Language** | Set the display language for the entire app. Changes take effect immediately. | English, Japanese |
| **Tool message style** | Control the tone of messages shown in the activity indicator while Claude works. **Funny** displays playful phrases like "Speed-reading..." while **Boring** shows literal descriptions. | Funny, Boring |
| **Skip delete worktree confirmation** | When enabled, deleting a worktree removes it immediately without a confirmation dialog. Keep this off if you want a safety prompt before deletion. | On, Off |

## Language behavior

Braid uses your browser locale on first launch, then stores your preference in local storage. You can also change the language from the **Theme** dialog accessible via the sidebar. Both English and Japanese translations cover all UI text including buttons, labels, dialogs, and status messages.

## Tips

- If you work in a team with mixed language preferences, each person can set their own language independently.
- The tool message style setting only affects the activity indicator during Claude sessions. It does not change Claude's responses or behavior.
