# About

The About page shows the Braid application identity, version, and mission.

## Open the About page

Go to **Settings > About**.

## Information displayed

- **Braid wordmark and version** - The current release version number.
- **Codename** - The version codename for the current release.
- **Tagline** - A brief description of what Braid does.

## Mission

A card describing the purpose and philosophy behind Braid.

## Capabilities

A list of the core capabilities Braid provides, such as worktree management, AI-powered coding, and integrated development tools.

## Creator

Attribution and build information at the bottom of the page.
