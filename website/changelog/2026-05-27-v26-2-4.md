A maintenance release focusing on terminal readability and agent compatibility. Light theme users will notice better contrast in terminal output, and Codex sessions now correctly surface input notifications.

{/* truncate */}

## Fixed

- **Terminal contrast in light themes** - Adjusted ANSI color mappings so terminal text is easier to read when using light themes.
- **Codex terminal input notifications** - Fixed an issue where Codex agent sessions would not trigger input-needed notifications, causing prompts to go unnoticed.

## Removed

- **Spotify embedded app** - Disabled the Spotify webview integration due to persistent DRM permission issues. We may revisit this once upstream Electron Widevine support stabilizes.
