You can now use the full 1M context window. Toggle it right from the model selector dropdown - a "1M" badge appears on the chip so you always know when it's active. Opus 4.6, Sonnet 4.6, and Mythos get native 1M support; older Sonnet models use the beta header automatically. Set your default in Settings > AI.

We also added trackpad swipe navigation. Two-finger swipe left or right over the center panel to cycle between tabs - sessions, files, changes, whatever you have open.

## Improvements

- Extended context (1M tokens) toggle in the model selector with automatic beta header handling for older models
- Two-finger trackpad swipe navigation between center panel tabs
- Project avatar visibility toggle in Settings > Appearance - hide sidebar project icons if you prefer a cleaner look
- Default extended context preference in Settings > AI

## Fixes

- Snippet and terminal cards no longer overflow to the left on narrow windows
- Toasts and sounds now fire correctly for AskUserQuestion and ExitPlanMode events
- Dock bounce now triggers for waiting_input even when Braid is focused
- Desktop notifications receive the correct reason (question/plan approval)
- Snippet cards got accent borders and pill-styled badges for better visual distinction
